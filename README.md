# Recivery

Recipe + Delivery

A Python-based application

1.  Recommend/Search recipes
2.  Recommend grocery stores based on US address


## Installation

```bash
pip install -r requirements.txt
python -m pip install lxml
```


## Usage

```bash
python3 main.py
```
